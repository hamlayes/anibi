GitHub



Github, Inc est une entreprise de développement et services logiciels sise aux États-Unis. Github développe notamment la plateforme Github, la structure Electron. Le 4 juin 2018, Microsoft annonce l’acquisition de l’entreprise pour la somme de 7,5 milliards de dollars américains. L'éditeur de texte Atom est abandonné.

Se Connecter



S'identifier ou se créer un compte pour pouvoir accéder à GitHub.


Projet



Après la connexion, on peut retrouver sur la gauche le bouton vert "new" où nous pouvons créer de nouveau projet.
En dessous il y a la liste des projet déjà créer.


Une fois que vous avez séléctionné un projet déjà créer, on devrait tomber sur cette page ci-dessous :



Ici, on est dans l'onglet où on peut voir les dossiers qui on était créer dans le projet


Dans les projets on à la possiblité de gérer l'avancer du projet grâce à l'onglet "project". Cela se présente sous forme de tableau diviser en 4 partie.




C'est ici que nous pouvons voir tous les fichiers qui constitue le programme du projet.




Possibilité d'ouvrir les scripts avec différent logiciels.
